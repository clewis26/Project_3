Project 3: Image Classification

Data Set

The data is obtained from the Kaggle competition website

Include "train_x.bin", "train_y.csv" and "test_x.bin" data files in the same folder as the mlp_final.py script

To run mlp_final.py

Include "train_x.bin" and "train_y.csv" data files in the same folder as the mlp_final.py script

python mlp_final.py

This will output training and validation accuracies based on our data

To run mlp_test.py

Include all data files from http://yann.lecun.com/exdb/mnist/ in the same folder as the mlp_test.py script

python mlp_test.py

This will output training and validation accuracies based on the MNIST data
